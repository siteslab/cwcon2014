DESCRIPTION
============

Provide views with new field that let's you add checkbox for selecting each row
Useful when you want to views as form for selecting data, specially in multipage forms

There is also a hidden text field that stores the selected rows' nids

INSTALLATION
============

Enable the module at Administer >> Modules.


USAGE
============

- Create a new view
- Go to the fields section and add "Global: Row Selector (Row Selector)"
